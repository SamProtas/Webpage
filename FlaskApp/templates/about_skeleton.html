About the webpage



Languages Utilized

  • Python - API scraping and all statistical work
  • SQL - Majority of information storage
  • Flask - web application framework for webdesign
  • Javascript - Minor usage for chart/graph generation
  • HTML / CSS - The rest of the static web design
  • Crontab - Daily automated data generation
  • R - Minor statistical analysis and practice

Project Explanation

  • First at foremost, everything on this webpage is automated rather than static displays of charts and calculations inputted into an html page. Everything has underlying python code running on a server to update the data as it updates in real life.
  • Started with Ubuntu server set up, a domain name, and getting my Flask environment online.
  • Dynamic analysis of instagram accounts. Pulled all information from an account called 'Hungrygrls' and organized it in databases for various graphical analysis. Additionally created automated datascrapes to track the followers over time every 30 minutes.
  • Stubhub analysis. Working to automate the pricing and posting of a set of season tickets I own.
  • Yahoo Developers API - Currently sifting through data provided by Yahoo Query Language to find trends in my two fantasy football leagues. Yahoo's organization of their JSON responses is making basic information slightly more difficult to grab than originally anticipated...
  • All of the charts presented throughout the webpage are product of Highcharts.com API
  • All html/javascript/css was recently done for a more simple/sleek design. This is still a work in process and will hopefully continue to improve over time. I rather focus on data analysis at the time being.
